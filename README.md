# Python PowerUp 🚀

Projeto desenvolvido para estudos de automação de tarefas utilizando Python.

## Tecnologias utilizadas
- Python
- PyAutoGUI
- Pandas

## Funcionalidades
- Automação de cadastro de produtos
- Manipulação de dados CSV
- Execução automática de tarefas repetitivas

## Objetivo
Aprofundar conhecimentos em automações, produtividade e integração de sistemas.
