# Python PowerUp 🚀

Projeto desenvolvido para estudos de automação de tarefas utilizando Python, PyAutoGUI e Pandas.

## 📌 Sobre o projeto

O objetivo deste projeto é automatizar o cadastro de produtos em um sistema web de forma prática e eficiente, simulando processos reais utilizados em ambientes corporativos.

A aplicação realiza:

- abertura automática do navegador;
- acesso ao sistema;
- autenticação/login;
- leitura de base de dados CSV;
- cadastro automatizado de produtos;
- preenchimento automático de campos;
- repetição automática do processo até finalizar a base de dados.

Este projeto simula processos repetitivos presentes em ambientes empresariais, buscando otimizar produtividade, reduzir tarefas manuais e demonstrar aplicações práticas de automação utilizando Python.
